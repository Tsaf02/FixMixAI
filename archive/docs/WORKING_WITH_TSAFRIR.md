## How to Communicate

**Language:**
- Tsafrir's native language is Hebrew. He understands English well.
- If writing in Hebrew — it is faster for him.

**CRITICAL — RTL/LTR mixing in responses in hebrew mixed with English:**
Hebrew text mixed with English terms in the same line causes the word order to
appear reversed on screen (the RTL mixing problem this project is solving).

When writing Hebrew responses that include English terms, ALWAYS use one of:
1. Put the English term on its own line (preferred)
2. Put the English term in parentheses or quotes within the Hebrew sentence
3. Never write an English word in the middle of a Hebrew sentence without separation

**Example of WRONG:**
"נריץ את הפקודה npm install כדי להתקין"

**Example of CORRECT:**
"נריץ את הפקודה הבאה:
npm install
כדי להתקין את הספריות."

## How to Give Instructions

- Give ONE step at a time with full details
- Warn about possible mistakes before they happen
- Wait for the user to confirm completion before giving the next step
- For complex tasks: state the complexity upfront, how many steps, and ask if ready to begin
- Never give a long numbered list of steps — he will get lost

**He prefers:**
- Brevity over completeness
- One best option over multiple choices
- To plan before executing
- to ask him anything if its not clear.
